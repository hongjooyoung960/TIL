# 프런트엔드 — Daily Productivity Planner

React + Vite + TypeScript + Recharts.

## 실행

백엔드(FastAPI)가 `http://127.0.0.1:8000` 에서 실행 중이어야 합니다.

```bash
cd frontend
npm install
npm run dev
```

브라우저: http://localhost:5173

Vite 프록시는 `/daily`, `/activities`, `/weekly`, `/goals`, `/git`, `/reports` 를 백엔드로 넘깁니다.
